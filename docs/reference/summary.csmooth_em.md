dimensions, initialization/adaptive settings, parameter ranges, last
trace values, and last-step relative changes.

``` r
summary(object, ...)
```

## Arguments

  A `csmooth_em` object.

- ...:

  Unused.

## Value

A list with class `"summary.csmooth_em"`.
